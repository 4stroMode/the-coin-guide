# The Coin Guide

The Coin Guide is a mod that adds a guide and community information to the levels with user coins, to make it easier to find them and have some feedback about the difficulty of the coins.

<cy>Features:</c>
* **Coin Guide Button on rated levels with user coins**
* **Coin Community Difficulty Rating System and Difficulty Coin indicator** 
* **Admins & Moderators roles that helps adding guides and rating coins in the levels**
* **Coin Advanced Search Page on the search browser page**

All the features sows only on rated levels that have user coins.
You can also help rating the difficulty of the coins with a button near the practice progress bar.
